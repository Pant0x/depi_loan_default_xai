LOAN XAI SYSTEM

Automated Explainable AI for Loan Default Prediction

Project Summary

This project implements an end-to-end automated machine learning pipeline for predicting loan defaults, with a strong emphasis on model explainability and interpretability. The system covers the full data science lifecycle — from exploratory data analysis and preprocessing through model training, hyperparameter tuning, and post-hoc explainability — producing transparent, human-readable credit risk assessments that financial institutions and individual applicants can understand and trust.

  • ⚡
    Multi-Model Training Pipeline: Trains and evaluates multiple classifiers including LightGBM (with SMOTE, tuning, and early stopping variants), XGBoost, Random Forest, Logistic Regression, Ensemble Soft Voting, and deep learning Tabular MLPs (Keras) — selecting the best-performing model via ROC-AUC, precision-recall, and confusion matrix analysis.
  • 📊
    SHAP & LIME Explainability: Generates SHAP waterfall charts, beeswarm plots, and feature importance rankings alongside LIME local surrogate models — providing both global and per-prediction explanations of how each feature (credit score, DTI ratio, interest rate, etc.) influences the default probability.
  • 🎯
    Automated Feature Engineering: Automatically computes 27+ engineered interaction features including log-scaled income/loan ratios, debt burden scores, risk composite indicators, employment ratios, and high-risk binary flags — transforming raw applicant data into a 36-dimensional feature space optimized for model performance.
